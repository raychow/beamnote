Mystique 主题已经于近期更新到 2.0.2 版本, 经过我的测试, 目前发现有如下更新:

## 一、重新翻译的简体中文语言包

此版本简体中文的翻译质量比上一版本有了明显提升, 大部分翻译错误均被纠正, 但仍存在少量问题.

## 二、设计面板不再支持 IE 浏览器

使用任何版本 IE 浏览器进入设计面板均会出现「这个设计面板不支持 IE」的提示, 可能主题的作者要和 IE 决裂了吧 :-).

如果您习惯使用 IE 浏览器, 可以返回 Mystique 1.72.

## 三、增加「打印文章」功能
<!-- more -->

经测试, Chrome 浏览器打印结果是空白页, 而 Opera 无法打印.

## 四、修改了 Mystique 的控制板界面

合并了两个模块, 调整了保存与恢复按钮的位置. 当然, 这不是重要更新.

## 五、可选择固定或自适应的页面宽度

无法定义固定宽度的大小, 而是采用 [960 网格系统](http://960.gs/).

相应的, 去掉了前台鼠标移动到右上角时显示的调整页宽的按钮.

## 六、新增、改变了多项自定义功能

1. 可定义在文章页、分类、归档与搜索页面显示的信息;
2. 精选文章是手动在文章编辑页面定义的, 可以选择出现在更多类型的页面, 并支持多选, 可以改变显示的数量与时间范围. 这里可能存在一个问题, 我使用的时候 404 页面总是提示没有精选文章;
3. 可关闭顶部导航;
4. 去掉了 CSS 代码的示例.

## 七、文件结构有较大改变

目前发现:

1. 新版本将很多分散在各页面的代码放置到统一的文件中;
2. 自定义的用户样式表不再直接嵌入网页, 而是调用动态生成的 CSS 文件.
总的来说, 这次更新无特别的亮点, 您可以自行考虑是否升级.
